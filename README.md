# Trabalho Individual - Programação Orientada a Objetos

Repositório destinado aos exercícios praticos de lógica e POO em Java.

Aluna: Vitória de Paula Zanchet.

## Exercícios Realizados:
- **Exercício 1 (Lógica):** Simulador de Caixa Eletrônico com menu 'do-while', controle de saldo, limite de valor por saque e controlador de saques diários.
- **Exercício 2 (Classes e Objetos):** Migração da lógica do caixa eletrônico para POO, utilizado atributos, métodos e bônus de boas-vindas no construtor.
- **Exercício 3 (Relacionamentos):** Modelagemde E-commerce com composição entre 'Cliente', 'Pedido' e 'ItemPedido', incluindo cálculo automático de frete.

## Tecnologias e Ferramentas:
- **Linguagem:** Java.
- **IDE:** IntelliJ IDEA.
- **Versionamento:** Git e GitHub (autenticação via PAT).
- **SO:** Linux (Zorin OS).
